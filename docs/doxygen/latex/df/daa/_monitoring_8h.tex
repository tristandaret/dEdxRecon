\doxysubsection{Detailed Description}
Monitoring utilities and entry points for analysis workflows. 

Declares functions used to run monitoring, produce diagnostic summaries (histograms, performance numbers) and control analysis settings.

A small executable {\ttfamily \doxylink{_analysis_8cxx}{Apps/\+Analysis.\+cxx}} exists which calls {\ttfamily \doxylink{namespace_reconstruction_aa6c4189b9f8794fb9b7ad71b9765b826}{Reconstruction\+::\+Monitoring()}} as a top-\/level entry point to run analysis workflows; the implementation of Monitoring lives in {\ttfamily \doxylink{_monitoring_8cxx}{Reconstruction/src/\+Monitoring.\+cxx}}.

The source file contains routines that assemble diagnostic histograms, run monitoring workflows and produce summaries for inspection. It is the primary entry point used by the {\ttfamily Analysis} executable to run automated monitoring and analysis sequences; configuration and selection setup live in the header-\/level declarations and other helper modules.
