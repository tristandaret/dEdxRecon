\doxysubsection{Detailed Description}
Declarations for cluster-\/level fitters used to fit pad-\/cluster charge distributions. 

This header defines both horizontal and diagonal cluster fitter classes and related helper functions to run fits on Samples, Events and Clusters. Implementations live in \doxylink{_cluster_fitter_8cxx}{Cluster\+Fitter.\+cxx}.

The corresponding implementation uses ROOT\textquotesingle{}s TVirtual\+Fitter/\+TMinuit and registers file-\/local static bridge classes that forward Minuit callbacks to the instance methods (Chi2 evaluation). The source contains both the horizontal and diagonal fitter implementations and several helper functions to apply fitters to Samples, Events and individual Clusters.
