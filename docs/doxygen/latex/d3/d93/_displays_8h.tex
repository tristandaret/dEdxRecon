\doxysubsection{Detailed Description}
High-\/level display utilities and plotting helpers. 

Declares functions and helpers used by the GUI and plotting components of the reconstruction pipeline (event displays, waveform displays, cluster visualisations, etc.).

The implementation in {\ttfamily \doxylink{_displays_8cxx}{Displays.\+cxx}} contains high-\/level presentation code that constructs ROOT-\/based canvases, histograms and graphical summaries. This is primarily display code that depends on ROOT and should remain separate from core reconstruction algorithms.
