\doxysubsection{Detailed Description}
Declarations for track reconstruction utilities and helpers. 

This header exposes the Track\+Recon top-\/level functions that run reconstruction over a \doxylink{class_sample}{Sample} or an \doxylink{class_event}{Event} using a \doxylink{class_track_fitter}{Track\+Fitter}. Implementation is in \doxylink{_track_recon_8cxx}{Track\+Recon.\+cxx}.

The implementation orchestrates fitting over Samples and Events and uses the \doxylink{class_track_fitter}{Track\+Fitter} API to run per-\/track minimisations. It contains the top-\/level control loops that increment fit/failure counters and report simple summaries after processing.
