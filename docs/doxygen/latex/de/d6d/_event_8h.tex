\doxysubsection{Detailed Description}
\doxylink{class_event}{Event} container holding modules, clusters and tracks. 

The \doxylink{class_event}{Event} class aggregates detector modules, clusters and track candidates for a single event. It also holds pointers to model objects (geometry, electronics, charge) used by simulation and reconstruction.
