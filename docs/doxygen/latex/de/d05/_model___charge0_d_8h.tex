Declares \doxylink{class_model___charge0_d}{Model\+\_\+\+Charge0D} which estimates integrated charge produced by an ionising particle in a pad or collection region. Implementation is in \doxylink{_model___charge0_d_8cxx}{Model\+\_\+\+Charge0\+D.\+cxx}.
