\doxysubsection{Detailed Description}
Miscellaneous reconstruction helper functions and utilities. 

Small helpers and adapters used across reconstruction, plotting and fitting code (linspace equivalents, I/O helpers, Bethe-\/\+Bloch wrappers, etc.).

The implementation file {\ttfamily \doxylink{_recon_tools_8cxx}{Recon\+Tools.\+cxx}} contains a collection of lightweight utilities relied upon by higher-\/level modules (correction functions, selection initialization, CSV helpers and physics wrappers).
