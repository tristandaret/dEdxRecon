\doxysubsection{Detailed Description}
Abstract base class for data uploaders. 

Declares the \doxylink{class_uploader}{Uploader} interface used to read input samples and produce {\ttfamily \doxylink{class_event}{Event}} objects for the reconstruction pipeline. Concrete uploaders implement format-\/specific logic.
