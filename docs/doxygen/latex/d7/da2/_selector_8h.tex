\doxysubsection{Detailed Description}
Selection utilities to filter events, modules and clusters. 

The \doxylink{class_selector}{Selector} class encapsulates selection stages and criteria and provides methods to apply selections to samples and events.

Implementation in {\ttfamily \doxylink{_selector_8cxx}{Selector.\+cxx}} contains the selection-\/stage definitions and concrete logic for known selection presets (DESY21, CERN22, etc.). Changing selection logic can affect downstream analyses; keep tests updated when criteria change.
