\doxysubsection{Detailed Description}
Combined fitting utilities declaration. 

Declares the combined\+Fit helper used to fit multiple TGraph\+Errors/\+TF1 series together (e.\+g., high-\/level fits like Bethe-\/\+Bloch curve fits).

Implementation coordinates multiple TF1 instances and ROOT fit helpers to perform simultaneous fits across different data series (for example, fitting Bethe-\/\+Bloch parameters across particle species). It uses ROOT\textquotesingle{}s fitting framework including Wrapped\+Multi\+TF1 and the ROOT\+::\+Fit\+::\+Fitter interface and exposes a single helper function {\ttfamily \doxylink{_combined_fit_8h_a7a2bd01b76d62b38240727b6c6ad12b0}{combined\+Fit}} which takes vectors of TGraph\+Errors and TF1 pointers and runs a global chi2 fit.
